# Info_Scan_Box
工具是融合了web_source_scan来整合的

你应该首先pip request模块

#### 工具介绍

##### 此工具是仿制各路大佬的工具，来制作的一款工具，期间借鉴了各路大佬的字典等等

- ##### 工具支持多线程（正在全力补全）

- ##### 支持网页源代码url提取

- ##### 支持目录扫描（有多款字典）

- ##### 支持子域名扫描

- ##### 支持黑名单（需要自己将黑名单加入）

##### 

#### 怎么使用？

你首先要将你目标url放入urls.txt,假如你想要使用黑名单，就需要将黑名单url的关键词放入black.txt
你假如想要使用crt爬取，你就首先要在 crt.sh 网站进行搜索，然后将其源代码放入read.txt
```shell
./Run.sh
```
