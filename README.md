# Info_Scan_Box
工具是融合了web_source_scan来整合的

#### 工具介绍（工具还在开发当中！！！）

##### 此工具是仿制各路大佬的工具，来制作的一款工具，期间借鉴了各路大佬的字典等等

- ##### 工具支持多线程（正在开发）

- ##### 支持nmap的简易化扫描（正在开发）

- ##### 支持网页源代码url提取

- ##### 支持目录扫描（有多款字典）

- ##### 支持子域名扫描（正在提高效率）

- ##### 支持黑名单（需要自己将黑名单加入）

##### 

#### 怎么使用？

```shell
./Run.sh
```
